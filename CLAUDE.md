# CLAUDE.md

## Commands
- `npm start` - Start dev server
- `npm test` - Run tests (Vitest + React Testing Library)
- `npm run build` - Production build
- `npm run typecheck` - Type check JS files with TypeScript

## Tech Stack
- React 19 (functional components, hooks)
- Vite (build tool)
- Vitest (test runner)
- TypeScript (type checking via checkJs)
- Plain CSS (no preprocessor)
